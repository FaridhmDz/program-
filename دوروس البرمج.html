HTML

HTML (لغة توصيف النصوص التشعبية) هي الهيكل الأساسي لصفحات الويب.

مثال بسيط:

<!DOCTYPE html>
<html>
<head>
    <title>مثال HTML</title>
</head>
<body>
    <h1>مرحبًا بالعالم</h1>
    <p>هذه فقرة في HTML.</p>
</body>
</html>

CSS

CSS (أوراق الأنماط المتتالية) تستخدم لتنسيق وتجميل صفحات الويب.

مثال بسيط:

body {
    background-color: lightblue;
}
h1 {
    color: navy;
    text-align: center;
}
p {
    font-family: verdana;
    font-size: 20px;
}

JavaScript

JavaScript هي لغة برمجة تستخدم لإضافة تفاعلات وديناميكية إلى صفحات الويب.

مثال بسيط:

document.addEventListener('DOMContentLoaded', function() {
    document.querySelector('h1').textContent = 'تم تغيير النص بواسطة JavaScript!';
});
